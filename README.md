# STOCK-MANAGEMENT-SYSTEM
Overview :-
The Stock Management System is a command-line interface (CLI) application written in C++ designed to efficiently manage a stock inventory. The system allows users to perform essential operations such as adding, updating, removing, and viewing stock items, making it ideal for small businesses, warehouses, or any inventory management process.

Features :-

    Add New Stock Items: Allows the user to add new products to the inventory with relevant details like name, category, price, and quantity.
    
    Update Stock Information: Modify existing stock details such as price or quantity based on real-time requirements.
    
    Remove Stock Items: Enables the deletion of products from the inventory if they are no longer required.
    
    View Inventory: Provides an option to display the complete list of current stock items, showing their details.
    
    Search Stock: Enables the user to search for stock items based on various criteria, such as product name or category.
    
    Sort Inventory: Sort stock items by price or quantity, making it easier to analyze inventory.

Technologies Used :-

    C++: The core programming language used for implementing the project.
    
    Data Structures: The project utilizes various data structures like arrays, vectors, and lists to manage inventory items efficiently.
    
    Terminal Mode: The application is designed to run in the terminal/command-line interface.
